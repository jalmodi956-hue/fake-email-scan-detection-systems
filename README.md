# Fake Email Scan Detection System

## 📌 Project Overview

Fake Email Scan Detection System is a cybersecurity project developed using Python and Flask. It helps users identify suspicious or phishing emails by analyzing the sender's email domain, email content, suspicious keywords, and URLs. The system calculates a risk score and classifies the email as Safe, Suspicious, or Phishing.

---

## 🚀 Features

- Email Domain Verification
- Trusted Domain Detection
- Suspicious Keyword Detection
- URL Detection
- Risk Score Calculation
- Safe / Suspicious / Phishing Verdict
- Scan Date & Time
- SQLite Database Storage
- Dashboard with Statistics
- Pie Chart Visualization
- Last 10 Scan History
- Export Scan History to CSV
- Download PDF Report
- Print Report
- Responsive Bootstrap User Interface

## ▶️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Fake_Email_Detector.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python app.py
```

4. Open in your browser

```text
http://127.0.0.1:5000
```
